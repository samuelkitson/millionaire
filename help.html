Introduction

Welcome to the 'Who Wants to Be a Millioniare? Quiz Edition - Official Help Guide'! In this simple and easy-to-read guide, you'll quickly find out how to set up and play the game in a variety of different modes. We hope you enjoy playing and we would welcome any suggestions for new catgories and questions as well as improvements to the current database (please see below). Thank you!

Please note: this game will not work in Internet Explorer. For best experiences, use Google Chrome.

Setup for multi-screen mode

Outlined below are the steps for setting up the game. We've aimed to make this setup process as easy and as quick as possible. The system will remember your selected topic between rounds to help you speed up gameplay.

  1. Choose how you want to play. If you have 2 screens (includes projectors) conencted to your computer, follow the instructions under 'Setup for multi-screen mode' below. If you have one screen, just open this page and leave the 'enter code' box blank.
  2. Select the question set you want to play with by using the dropdown menu near the right.
  3. Click 'sync pages' to initialise and then click 'start game' to begin.
  4. Load the first question when you're ready by clicking 'load next question'.

Setup the game

With this game, you have the option to play exactly like on TV - if you have 2 screens connected to your computer (includes projectors), you can use one to control the game and the other fo the contestant to use. The contestant won't be able to control the game and they won't be able to see the controller's button and options. This is more realistic and more like the TV show.

  1. To set this up, you must configure your PC to 'extend screen' mode. In Windows, press the Windows key and P at the same time. Then select 'Extend'.
  2. Open your web browser twice (preferably Google Chrome). Drag one onto the other screen and keep the other on the first screen. Maximise each one so that they fill their screen.
  3. On your screen, open this page.
  4. On the other screen (the contestant's one), open this page.
  5. Click on each page in turn and press F11 to make it full screen.

Basic controls

Read the following points to learn the basic methods of controlling the game. These will all work the same way, whether you're in single-screen or multi-screen mode.

  1. Load the question by clicking the green 'load next question' button.
  2. If you've already had this question, click the orange 'generate new question' button to load another one.
  3. If the player asks to use a lifeline, click the corresponding icon. See 'Lifelines' below to find out how this works.
  4. If the player wants to leave with the money they've got, click the red 'leave with the money' button (not available on the first question). A popup will appear displaying their score.
  5. When the player announces their final answer, click the corresponding answer box. It will turn orange and the music will change.
  6. Click the blue 'reveal answer' button to make the correct answer turn green.
  7. If they got it right, the 'load new question' button will show. If they got it wrong, a popup will appear displaying their score.
